# react-native-tab-view-easy

react-native-tab-view-easy

## Installation

```sh
yarn add react-native-tab-view-easy
```
OR
```sh
npm install react-native-tab-view-easy
```

## Usage

```js
import { TabView, Tab } from 'react-native-tab-view-easy';

// ...

<TabView>
  <Tab title={'Tab 1'}>
    <Text>Content</Text>
  </Tab>

  <Tab title={'Tab 2'}>
    <Text>Content</Text>
  </Tab>
</TabView>
```
